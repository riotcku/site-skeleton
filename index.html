Site Skeleton
This is mostly a collection of useful css classes I keep using in every site.
The class 'main-container' of the outer container defines the horizontal max width of the content, while 'content-container' defines space between content.
The class 'column-container' defines flex layout as `column`.
In the same vein, 'row-container' defines flex layout as `row`. Note the use of `space-between` class to define `justify-content` of the flexbox.
